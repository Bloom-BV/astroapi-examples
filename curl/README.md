# cURL Examples

## Prerequisites

- [curl](https://curl.se/) (usually pre-installed)
- [jq](https://jqlang.github.io/jq/) for JSON formatting
- An AstroAPI key

## Setup

```bash
export ASTROAPI_KEY="your-api-key-here"
```

## Run

```bash
bash curl/01-quickstart.sh
```
