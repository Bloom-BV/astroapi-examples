# JavaScript Examples

## Prerequisites

- Node.js 18+ (for native `fetch`)
- An AstroAPI key

## Setup

```bash
export ASTROAPI_KEY="your-api-key-here"
```

## Run

```bash
node javascript/01-quickstart.mjs
```
