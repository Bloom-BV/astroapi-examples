# PHP Examples

## Prerequisites

- PHP 8.0+ with curl extension (usually included)
- An AstroAPI key

## Setup

```bash
export ASTROAPI_KEY="your-api-key-here"
```

## Run

```bash
php php/01-quickstart.php
```

## Examples

| File | Description |
|------|-------------|
| `01-quickstart.php` | Your first API call |
| `natal-chart.php` | Full natal chart |
| `compatibility.php` | Love compatibility |
| `horoscope-daily.php` | Daily horoscope |
| `numerology-profile.php` | Numerology profile |
| `geocoding.php` | Location search |
| `chart-image.php` | Download chart as PNG |
